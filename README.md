# 📊 Pipeline de Geração de Dados - Vendas (Atacado & Varejo)

Este projeto simula um pipeline de dados automatizado, gerando informações fictícias de vendas e armazenando em um banco de dados na nuvem (Supabase), com execução diária via GitHub Actions.

---

## 🚀 Objetivo

Simular um ambiente real de dados para:

- Testes de Machine Learning
- Construção de dashboards (Power BI / Qlik)
- Simulação de Data Warehouse (Dimensão + Fato)
- Prática de Engenharia de Dados

---
